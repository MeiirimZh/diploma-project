# Дипломный проект
Мобильное приложение для повышения эффективности процесса обучения.
# Возможности
- Создание заметок
- Генерация конспектов через локальную модель ИИ
- Создание флеш-карточек
- Создание и прохождение тестов
- Статистика обучения
- Поддержка метода интервального повторения
- Импорт и экспорт заметок, наборов флеш-карточек и тестов
# Использованные технологии
- React Native
- Expo
- SQLite
